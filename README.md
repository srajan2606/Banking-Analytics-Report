# Banking Analytics Report

This repository contains an interactive **Power BI** dashboard designed to analyze banking customer demographics, transaction volumes, and retention metrics.

## 📊 Dashboard Overview

The Power BI report is divided into two primary analytical tabs:

### 1. Banking Customers Analytics
This section provides a high-level overview of the bank's customer base and transaction activity. 
**Key Performance Indicators (KPIs) include:**
* **Total Customers:** The overall count of the customer base.
* **Monthly Transactions:** The volume of transactions processed on a monthly basis.
* **Accounts by Type:** The count of `Account_customer ID` categorized by specific account types.
* **Demographics:** A breakdown of the customer count by gender.

### 2. Cohort Analysis
This section focuses on customer lifecycle and retention trends.
* **Customer Cohort Diagram:** A visual representation tracking specific groups of customers over time to analyze long-term behavior and retention rates.

---

## 🛠️ Tools Used
* **Power BI:** Used for data modeling, visualization, and interactive dashboard creation.

## 🚀 How to View the Report
1. Download the `.pbix` file from this repository.
2. Open the file using [Power BI Desktop](https://powerbi.microsoft.com/desktop/) to interact with the dashboards and explore the data.
